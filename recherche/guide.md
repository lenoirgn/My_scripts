# Mode d'Emploi Rapide

## Placement
Assurez-vous que les deux fichiers scripts (`recherche.sh` et `recherche_text.sh`) sont dans le même dossier.

## Permissions
```bash
chmod +x recherche.sh
```

## Chargement
```bash
source recherche.sh
```

## Lancement
```bash
recherche [Chemin/du/Fichier] [Type] [Chaîne]
```

### Exemples
```bash
recherche monFichier.log mot de
```

```bash
recherche /home/user/doc.txt caractere z
```

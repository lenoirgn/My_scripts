
# My_scripts

Bienvenue sur **My_scripts** !

Ici, tu trouveras principalement des exercices pour débutant en scripting. J’utilise ce dépôt pour apprendre, progresser petit à petit, et m’exercer sur des scripts simples. Ce n’est pas encore un ensemble de scripts professionnels ou très utiles… mais un jour, j’espère pouvoir y mettre de “vrais” scripts pratiques et efficaces !

## Pourquoi ce dépôt ?

- S’exercer et apprendre les bases du scripting (Bash, Python, etc.)
- Garder une trace de ma progression
- Revenir sur mes erreurs ou mes réussites pour m’améliorer
- Éventuellement, partager des astuces ou scripts plus utiles au fil du temps

## Comment utiliser ces scripts ?

Si tu débutes aussi, tu peux t’inspirer de mes exercices. Voici comment faire :

1. **Clone le dépôt :**
   ```bash
   git clone https://github.com/lenoirgn/My_scripts.git
   ```
2. **Rends les scripts exécutables si besoin :**
   ```bash
   chmod +x mon_script.sh
   ```
3. **Exécute-les via le terminal en fonction du langage :**
   - Pour un script Bash :
     ```bash
     ./mon_script.sh
     ```
   - Pour un script Python :
     ```bash
     python3 mon_script.py
     ```

Lis bien les commentaires dans chaque fichier, ils pourront t’aider à comprendre la logique.
